picGo

配置使用picGo

下载picGo

地址：https://github.com/Molunerfinn/PicGo/releases

获取Token

1.在github 中点击头像中的Settings

2.选择Developer settings

3.点击Personal access tokens

4.在其中点击 Generate new token

5.输入密码验证后 ，在Note中写tokens的描述，Expiration 设置过期时间 ，选择No expiration (永远不过期)

选中repo选项。

6.点击下方绿色按钮 Generate token 即可获取到 Tokens

注意 Tokens 只显示一次 一定要另外保存 配置picGo是需要填入 Tokens

配置picGo

填写完毕后设置为默认图床。

Typora配置

在Typora文件 下的偏好设置中 设置图像

上传服务 选择 picGo(app)

PicGo路径 就是 PicGo文件所在的位置。

设置完毕即可使用。粘贴图片后，点击上传图片，自动就会上传到gihub仓库中。
